A sophisticated forensic tool developed for elite Discord community administrators. This engine is specifically engineered to trace the origins of invite links and expose "Ghost Advertisers" who use alternative accounts to bypass server security.

## 🛡️ Forensic Use Case: Exposing the "Real" Creator
In large communities, malicious users often use **Alt Accounts** (Fake Accounts) to DM-advertise. However, they frequently use **Permanent Links** generated by their **Main Account**. 

**Forensic Engine** decodes the link's metadata to reveal the true identity of the creator, effectively unmasking the person behind the campaign.

## 🧬 Engine Intelligence Matrix

* **`/check` [Link Fingerprinting]:** * Extracts hidden server metadata (Owner ID, Verification level, Boosts).
    * Validates link integrity and calculates precise expiration.
* **`/checkuser` [Node Analysis]:** * Maps mutual server nodes between the target and the forensic scraper.
    * Generates a full **Permission Matrix** (Admin, Manage Server, etc.).
    * Scans for Ownership Signals across shared networks.
