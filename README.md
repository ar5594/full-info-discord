# 🔍 Forensic Engine | Advanced Discord Intelligence Tool

A professional-grade digital forensic and metadata extraction bot built with **Discord.js v14**. This tool is designed for deep server analysis and user permission auditing.

---

## 🚀 Key Features

* **Link Trace (`/check`)**: 
    * Extract server metadata from any invite link (Owner ID, Creation Date, Verification Level).
    * Audit link integrity and precise expiration countdowns.
    * Track population metrics (Total vs. Online members).
* **User Radar (`/checkuser`)**:
    * Deep-scan user profiles using high-precision forensic methods.
    * Map mutual server networks and identify owned guilds.
    * Full Permission Matrix: View detailed administrative and moderator rights across shared nodes.
* **Premium Interface**: Uses a "Dark Mode" aesthetic with interactive menus and buttons for seamless navigation.
