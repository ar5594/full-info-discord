# 🔍 Forensic Engine v2.0 | Discord Security & Anti-Alt Radar

A high-precision forensic tool engineered for Discord community protection. This engine is designed to unmask "Ghost Advertisers" by tracing invite link metadata back to their original creators, exposing the main accounts behind fake advertisement campaigns.

---

## 🛡️ The Forensic Use Case: Exposing the "Real" Creator
In large communities, malicious users often use **Alt Accounts** (Fake/New Accounts) to spread DM ads. However, they frequently use **Permanent Links** generated by their **Main Accounts**. 

**Forensic Engine** decodes these links to reveal the true identity of the creator, effectively unmasking the person behind the campaign.

## 🧬 Core Intelligence Matrix

* **`/check` [Link Fingerprinting]:** * Decodes any `discord.gg` invite to find the **original creator's identity**.
    * Extracts server metadata: Owner ID, Creation Date, and Verification Level.
    * Validates link integrity (Permanency vs. Expiration).

* **`/checkuser` [Node Analysis]:** * Maps mutual server nodes between the target and the forensic scraper.
    * Generates a full **Permission Matrix** (Admin, Manage Server, Ban, etc.).
    * Scans for Ownership Signals and account timestamps.
